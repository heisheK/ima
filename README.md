# ima 多源资料整理 Skill 原型

这是一个用于腾讯 AI 产品训练营的交互网页原型，展示在 ima 首页新增「多源资料整理 Skill」入口，并进入支持多源导入、批量选择、AI 提炼和结果导出的工作台。

## 页面

- `index.html`：ima 风格首页，包含 Skill 入口。
- `skill.html`：多源资料整理 Skill 工作台。

## 交互亮点

- 支持公众号、论文 PDF、网页、已有笔记、会议纪要等多源资料入口。
- 支持多选、全选、反选和模拟导入。
- 支持生成笔记、综述、纪要、大纲、金句等输出结构。
- 支持将 AI 提炼卡片加入画布并拖拽重组。
